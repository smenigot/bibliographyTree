# Version 1: Graph Notebooks

This folder contains the notebook-based material originally prepared for bibliographic graph exploration.

## Contents

- `arbre_genealogique_papier_avec_filtrage.ipynb`: builds a historical graph around a target article while filtering journals by category.
- `arbre_genealogique_papier_sans_filtrage.ipynb`: builds the same type of graph without journal-category filtering.
- `recherche_articles_par_motscles_et_par_categories.ipynb`: explores article collection from keyword and category constraints.
- `exemple_graphe.png`: illustration used in the documentation.

## Purpose

Version 1 is notebook-first. It is useful for exploratory work, visual inspection, and method demonstration before moving to the more automated pipelines provided in versions 2 and 3.

## Notes

- These notebooks are the Git-ready copies of the original work.
- The committed image is kept only as documentation support.
- No generated notebook side-products are stored in the repository.
